# Context-Aware Multi-Tool Chatbot

An AI-powered chatbot built using Streamlit and Groq API.

## Features
- Conversational AI chatbot
- Streamlit-based web interface
- Modular backend architecture
- Secure API key management using .env
- OpenAI-compatible Groq integration

## Tech Stack
- Python
- Streamlit
- Groq API
- OpenAI SDK
- dotenv

## Future Improvements
- Conversational memory
- SQLite integration
- Multiple AI tools/APIs
- Vector database support
- React frontend
- FastAPI backend

## Run Locally

Install dependencies:

pip install -r requirements.txt

Run the app:

streamlit run app.py
